## 🎧 Usage Guidelines

These plugins work like physical gear. You use them on individual tracks or on your master output. 

*   Equalizers: Use these to change the tonal balance of a track. Cut low frequencies to remove mud or boost high frequencies to add shimmer.
*   Compressors: Use these to control the dynamic range. Quiet parts get louder, and loud parts get quieter.
*   Saturation: Use these to add character and warmth to digital recordings.

Start with the preset library. Presets offer starting points for different types of instruments, such as drums, vocals, or guitars. Load a preset and adjust the main knobs to taste.
